Integrates QuadMenu Mega Menu with the Avada theme. Requires QuadMenu and Avada.

== Description ==
[Demo](http://quadmenu.com/avada/) | [Premium](http://quadmenu.com) | [Documentation](http://quadmenu.com/documentation/) | [Mega Menu](https://wordpress.org/plugins/quadmenu/)

Unlike other mega menu plugins, QuadMenu has been designed for developers and regular users, as its offer additional hooks and filters that make it incredibly easy to integrate with your themes from the start.

This plugin called QuadMenu Avada allows you to easily integrate this mega menu with Avada, as it adds all the necessary settings to match the style and the layout of Avada theme.

If you're running a site with this theme, you can install this plugin a get an instant integration between your theme and QuadMenu.

== QuadMenu PRO ==

We also must stand out some additional features that are only available in the premium version like the integration with the customizer dashboard. This is an important feature if you will include this plugin in your themes project, as it allows you and your end users to tweak all menu color scheme or layout settings and see the effect of that changes in real time.

Also, we've included some incredible elements that will help you to stand out your theme, like the tabs menu, log-in and register form, social icons, and carousel menu.

== Installation ==

1. Go to the Plugins page in WordPress
1. Search for "Avada Mega Menu"
1. Click "Install"

After installation go to Appearance > Menus and create your mega menu.
